O que são Tags?

(Voltar para o início)

As Tags são atributos que formata a base de um site, como o texto, imagem, paragrafo, linkagem, entre outros que são essenciais.

Lembrando que Tags são apenas o que formata, para dar contexto e aparência de um site é preciso o CSS

Sempre seguem um formato padrão, que é ter a tag de início e a de fim.

Sendo assim, existe vários atributos que podemos colocar para formatar um site. Assim como as descritas em sequencia as que são basicas e muito utillizadas.

Principais Tags?

(Voltar para o início)
  • h1 até h6: define o título e os subtítulos
  • p: parágrafo
  • mark: marca texto
  • small: texto pequeno
  • strong: negriita o texto
  • i: deixa o texto em itálico
  • u: sublinha o texto
  • ol: cria uma lista ordenada
  • ul: cria uma lista não ordenada(mostrando caractéres ao invés de números)
  • li:constitui a lista
  • a: adiciona um link
  • hr: adiciona uma linha no site para organização ou separação
  • sub: deixa o texto ou número abaixo
  • sup: eleva o texto
  • blockquote: usado para citações

Quais não são muito utilizadas atualmente?

(Voltar para o início)

As Tags que não são utilizadas atualmente a não ser em sites muito antigos são target="_parent" e target="_top"
